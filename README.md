# CodeRetain

**每日 5 分钟编程练习，帮你巩固技能。**

专为希望在 AI 时代保持竞争力的开发者设计。涵盖 7 个技术分类，95 道精选题目。

## 功能

- 📝 每日 5 题挑战，保持编程手感
- 🔄 错题优先复习（间隔重复算法）
- 📊 30 天连续打卡统计
- 🎯 5 个技术分类 + Docker + JavaScript
- 🌐 中英文双语界面 + 双语题目独立切换
- 🏆 Streak 连续天数追踪
- 🎚️ 三级难度可选（初级/中级/高级）

## 技术栈

- Expo 56 / React Native 0.85
- React Navigation 7 (Bottom Tabs)
- TypeScript
- i18next (国际化)
- IndexedDB (Web 端存储)

## 运行

```bash
# 安装依赖
npm install

# Web 预览
npx expo start --web

# 或使用打包脚本
npm start
```

访问 http://localhost:8081

## 构建与部署

```bash
# 构建 Web 静态文件
npm run build:web

# 构建产物在 dist/ 目录
```
